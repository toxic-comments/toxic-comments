# Как пользоваться dvc

Перед началом работы - скачать актуальные данные (вместе с актуальным репозиторием)
git pull
dvc pull

Если были добавлены новые данные (или что-то изменено), то надо запушить изменения:
1) dvc add "data/interim/название_файла" . Эта команда создаст в той же директории файл название_файла.dvc с метаданными, которые надо закоммитить на гит. Другой разработчик сможет по этим метаданным скачать новые данные из хранилища. 
2) git add "data/interim/название_файла.dvc" и 
   git commit -m "Add new data"
3) git push и dvc push
